# Python Assignment_1- Control Structures

This repository contains two Python programs based on Control Structures .
--------------

## Task 1: Check if Number is Even or Odd
- Takes an integer from the user.
- Checks if the number is even or odd using an if-else condition.
- Prints the result.


--------------

## Task 2: Sum of Numbers from 1 to 50
- Uses a for loop.
- Calculates the sum of integers from 1 to 50.
- Prints the final sum.
